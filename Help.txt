Добро пожаловать в приложение "Classifier"! Это приложение предназначено для классификации объектов с использованием двух алгоритмов: перцептрона и классификатора минимального расстояния, использующий принцип поиска ближайшего соседа.

В этом разделе вы найдете подробную инструкцию по использованию приложения.

 - Загрузка данных

Чтобы загрузить данные, необходимо выбрать пункт "Open" в меню. В открывшемся окне выберите файл Excel, содержащий данные для классификации. Вы можете выбрать файл с расширением .xls, .xlsx или .xlsm. После выбора файла нажмите кнопку "Open". Данные будут загружены в таблицу на основной форме приложения.
Приложение может обработать от 1 до 5 параметров.

Чтобы увидеть характеристики вашей выборки нужно нажать на кнопку "Characteristics". После нажатия в ссответствующих таблицах появятся основные значения классов: среднее и дисперсия, так же отобразится внутриклассовое расстояние и межклассовое расстояние.

 - Классификация объектов

Для классификации объектов необходимо выбрать один из двух алгоритмов: перцептрон или классификатор минимального расстояния. Вы можете выбрать алгоритм.

 - Перцептрон

Перцептрон - это нейронная сеть, использующаяся для классификации объектов. Для использования перцептрона необходимо обучить его на обучающей выборке. Для этого нажмите кнопку "Train" на форме. После обучения в соответствующей таблице будут указаны значения весов соответствующих параметров вашей выборки.

 - Классификатор минимального расстояния

Классификатор минимального расстояния - это алгоритм классификации, основанный на вычислении расстояния между объектами. Для использования классификатора минимального расстояния не требуется обучение. Классификация происходит по принципу поиска ближайшего соседа, если расстояние между классифицируемым объектом и объектом с известным классом становится минимальным, то объекту присваивается класс этого известного объекта.

Для начала классификации нужно нажать кнопку "Go to predict", после этого откроется форма, где находятся поля, куда вы можете ввести значения параметров объекта. Далее Вы можете нажать кнопку "Predict by Perceptron" или "Predict by Min Distance" в соответствии с тем, с помощью какого метода желаете провести классификацию. После нажатия в соответствующем поле под кнопкой выйдет предсказанное значение класса для Вашего объекта.

 - Оценка качества классификации

Для оценки качества классификации вы можете использовать кнопку "Quality assessment" на форме. В соответствующих таблицах появятся результаты вычисления метрик качества классификации.

 - Закрытие приложения

Для закрытия приложения нажмите кнопку "Exit" в меню.
